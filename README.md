# Genatio

Open source grants. Verified on-chain. Powered by GenLayer Intelligent Contracts.

**Live Demo:** https://www.genatio.xyz

---

## What is Genatio?

Genatio is a trustless grant app for open source builders. Every project is verified autonomously by GenLayer Intelligent Contracts through Optimistic Democracy. No human approves or rejects anything. The AI and the chain decide everything.

---

## How It Works

**Submit** — Connect your wallet and submit your open source project with a GitHub URL, story, funding goal, and campaign duration.

**Verify** — A GenLayer Intelligent Contract fetches your GitHub data, scores your project, and lists it if it passes. No human involved.

**Fund** — The community funds active projects directly on-chain.
