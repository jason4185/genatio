# Genatio

Trustless open source grants powered by GenLayer Intelligent Contracts.

## What is Genatio?

Genatio is a decentralized grant platform where open source projects apply for funding and are verified entirely on-chain by AI — no admins, no committees, no middlemen. GenLayer Intelligent Contracts autonomously read GitHub repositories, evaluate project legitimacy, and decide grant eligibility through Optimistic Democracy.

**Live:** https://www.genatio.xyz
**GitHub:** https://github.com/jason4185/genatio

---

## How It Works

1. **Submit** — A builder submits their open source project with a GitHub repo, live URL, and funding goal.
2. **Verify** — A GenLayer Intelligent Contract fetches the GitHub API, reads the README and commit history, and scores the project 0–100. Projects scoring 40+ are listed.
3. **Fund** — The community funds active projects. When a campaign ends, funds are released — or flagged projects are re-investigated by the dispute contract.
